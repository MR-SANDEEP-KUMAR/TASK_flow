# TaskFlow – Team Task Management System

## 📌 Overview

TaskFlow is a full-stack web application that allows teams to manage projects and tasks with role-based access control (Admin/Member).

---

## 🚀 Features

* User Authentication (Login/Register)
* Role-based access (Admin & Member)
* Project Management
* Task Assignment & Tracking
* Dashboard Overview

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB (Atlas)
* API: REST APIs

---

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/MR-SANDEEP-KUMAR/taskflow.git
cd taskflow
```

### 2. Backend Setup

```bash
cd server
npm install
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret
```

Run backend:

```bash
npm start
```

---

### 3. Frontend Setup

```bash
cd client
npm install
npm start
```
